# 📊 HR Analytics – Employee Attrition Dashboard

An interactive HR Analytics Dashboard built using Power BI to analyze employee attrition patterns and uncover key factors influencing workforce retention.

---

## 📌 Project Overview

This project focuses on analyzing employee data to identify trends, patterns, and key drivers behind attrition. The dashboard transforms raw HR data into meaningful insights that help organizations make data-driven decisions to improve employee retention.

---

## 🎯 Business Objective

- Analyze employee attrition trends across departments and demographics  
- Identify key factors contributing to employee turnover  
- Evaluate the impact of salary, job role, and work conditions on attrition  
- Enable HR teams to take proactive actions to reduce attrition  

---

## 📊 Key Performance Indicators (KPIs)

- Total Employees  
- Total Attrition  
- Attrition Rate (%)  
- Average Salary  

---

## 🛠 Tools & Technologies Used

- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query (Data Cleaning & Transformation)  
- Data Modeling  

---

## 📐 Data Modeling

- Cleaned and transformed raw HR data using Power Query  
- Built a structured data model for efficient analysis  
- Created relationships between tables  

---

## 📈 Key Insights

- Certain departments show higher attrition rates  
- Employees with lower salaries are more likely to leave  
- Younger employees show higher attrition trends  
- Job roles and overtime significantly impact attrition  
- Work-life balance and job satisfaction influence retention  

---

## 📊 Dashboard Features

- KPI cards for quick overview  
- Department-wise attrition analysis  
- Gender and age-based segmentation  
- Interactive slicers for filtering  
- Clean and user-friendly design
